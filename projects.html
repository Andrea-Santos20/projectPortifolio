Sobre meus projetos

Os projetos foram desenvolvidos como forma de aprendizagem e fixação de conhecimentos.

Mario Jump é um projeto bem simples, com as liguagens HTML, CSS e JavaScript.

Rick And Morty é um projeto para aprendizagem de consumo de API externa, foi desenvolvido com a liguagem JavaScript, React e Axios para realizar a aquisição.

A Digital Car foi o projeto final do curso fullstack, Certified Tech Developer-CTD da Digital House. Foi um projeto realizado em grupo, muito completo, nos arriscamos a usar designs diferentes do que nos foi proposto e entregamos com qualidade, desenvolvi, com o auxílio de alguns colegas, a parte do front-end, utilizamos a liguagem JavaScript, React.

Esse foi um projeto desafiador, através dele alcancei o objetivo, de desenvolver uma aplicação, trabalhar em grupo e aprender mais.
